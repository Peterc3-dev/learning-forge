# 🔥 Learning Forge

Where raw curiosity gets hammered into capability.

This isn't a tutorial collection. It's a living document of applied learning — every challenge attempted, every concept wrestled with, every connection made between theory and muscle memory.

---

## Philosophy

Most people learn things and forget them. The ones who don't forget are the ones who *built something* while learning. This repo exists to close the gap between consuming knowledge and owning it.

Every entry here follows one rule: if I can't explain what I did and why it mattered, I didn't learn it yet.

## How to Read This Repo

Each folder contains its own README.md explaining what's inside and where things stand.

Entries are written as build journals — not polished tutorials, but honest documentation of the process: what worked, what didn't, what clicked, and what's still unclear.

Progress is tracked per-section. Some areas are deep, some are just getting started. That's the point — this is a forge, not a museum.
