# Cybersecurity

Offensive security learning, defense concepts, and real-world vulnerability hunting.

## Current Path

OverTheWire (foundations) → PortSwigger Labs (web app security) → Bug Bounty Programs (applied hunting)

## Why This Order

You can't hunt what you can't see. OverTheWire teaches you how systems actually behave under the hood. PortSwigger teaches you where web apps break. Bug bounties are where both converge on real targets with real consequences.

## Sections

### /overthewire/

Wargame progressions. Each level gets a write-up: what the challenge was, how I approached it, what concept it reinforced, and any connections to other domains.

### /ctf-writeups/

Capture The Flag event documentation. Challenge name, category, solution path, and lessons extracted.

### /bug-hunting/

Reconnaissance methodology, target notes, and findings documentation. Real-world application of everything learned in the training environments.

### /labs/

PortSwigger Web Security Academy, HackTheBox, and any custom lab environments. Structured progression through vulnerability classes.

---

## Key Insight

Pattern recognition transfers across domains. The same instinct that finds race conditions in a bus fare app finds IDOR vulnerabilities in web applications. Document the pattern, not just the solution.
