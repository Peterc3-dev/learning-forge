# Linux Sysadmin

System administration knowledge earned through 18+ years of daily driving Linux — configs, optimizations, and real problems solved.

## Primary Environment

- **OS:** CachyOS (Arch-based, performance-optimized)
- **Hardware:** GPD Pocket 4 (AMD HX 370)
- **Use case:** Daily driver, development, security research, local AI deployment

## Sections

### /cachyos/

Configuration files, optimization tweaks, and setup documentation for running CachyOS as a primary operating system.

### /troubleshooting/

Real problems encountered and solved. Each entry documents: the symptom, the diagnosis process, the fix, and what it revealed about the underlying system.

---

## Philosophy

The best sysadmin knowledge comes from things going wrong. Every resolved issue is a mental model strengthened.

Document the diagnosis process, not just the fix — the reasoning is more transferable than the command.
