# Gamified Courses

Custom learning tools and RPG-style progression systems designed around how my brain actually retains information.

## The Problem

Traditional tutorials teach concepts in isolation. I forget them within a week because there's no application context, no stakes, no feedback loop that matches my processing speed.

## The Solution

Build learning experiences that function like games — with progression systems, visual feedback, challenge scaling, and immediate application. Not gamification as decoration, but gamification as architecture.

## What Lives Here

- Personalized walkthrough systems (built with AI collaboration)
- Game-based learning experiments
- Progression tracking tools
- Any educational framework designed to bridge the gap between "I read about it" and "I can do it cold"

## Design Principles

1. **Immediate feedback** — every action produces a visible result
2. **Progressive difficulty** — challenge scales with capability
3. **Context-rich** — concepts are learned inside realistic scenarios, not abstract exercises
4. **Retention through application** — if it can't be applied within the session, it's not ready to be learned yet
